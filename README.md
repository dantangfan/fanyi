linux命令行下翻译工具

## Install

```
wget https://raw.githubusercontent.com/dantangfan/fanyi/master/fanyi.py
sudo mv ./fanyi.py /usr/local/bin/fanyi
sudo chmod +x /usr/local/bin/fanyi
```

## Usage

English To Chinese

```
$ fanyi test
###################################
#  test 测试 (U: tɛst E: test )
#  n. 试验；检验
#  vt. 试验；测试
#  vi. 试验；测试
#  n. (Test)人名；(英)特斯特
###################################
```

Chinese To English
```
$ fanyi 测试
###################################
#  测试 test (Pinyin: cè shì )
#  [试验] test
#  measurement
###################################
```
